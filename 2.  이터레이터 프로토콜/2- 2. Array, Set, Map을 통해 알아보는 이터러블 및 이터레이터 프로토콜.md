### 2- 2. Array, Set, Map을 통해 알아보는 이터러블 및 이터레이터 프로토콜
    - Array를 통해 알아보자
```javascript
    const list = [1,2,3]
    for(var i = 0 ; i< list.length; i++) {
        log(list[i])
    }
```

- Set를 통해 알아보자
```javascript
    const str = 'abc'
    for(var i = 0 ; i< str.length; i++) {
        log(str[i])
    }
```

- Map를 통해 알아보자
```javascript
    const list = [1,2,3]
    for(const a of list) {
        log(a)
    }
```
