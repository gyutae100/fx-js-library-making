# 기본 range 함수

- range 실행 시 리스트를 바로 만든다.

```javascript
const range = (l) => {
  let res = [];
  let i = -1;
  while (++i < l) {
    console.log(i);
    res.push(i);
  }
};

log(range(5)); //0,1,2,3,4

var list = range(4);
log(reduce(add, list)); // 6
```

# 느긋한 range 함수

- 제너레이터 이용

```javascript
const L = {}
const L.range = function *(l) => {
  console.log('hi~') // 첫 .next()실행 시만 실행된다.
  let i = -1;
  while (++i < l) {
    console.log(i)
    yield i
  }
};

log(range(5)); //0,1,2,3,4

var list = L.range(4); // iterator를 반환한다.
log(reduce(add, list)); // 6, 리듀서는 이터러블을 받을 수 있다.
```
